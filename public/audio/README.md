# 効果音ファイル（public/audio/）

このフォルダに **mp3 の効果音**を置くと、対局中の各イベントで再生されます。
ファイルが無くてもエラーにはならず、その音だけ無音になります（後から置けば鳴ります）。

> 読み上げ（「ダイヤの8！」「あがり！」「残念！」「パス」）は録音ファイル不要です。
> ブラウザ内蔵の音声合成（Web Speech API）で動的に発声します（`src/lib/audio/speech.ts`）。

## 必要なファイル名（固定）

| ファイル名 | 鳴るタイミング | 備考 |
|---|---|---|
| `shuffle.mp3` | 配札（ゲーム開始時） | シャッフル音 |
| `play.mp3` | カードを場に出したとき | 短いクリック/カチッ系 |
| `pass.mp3` | パスしたとき | 短い通知音 |
| `applause.mp3` | 誰かが上がったとき | 拍手 |
| `eliminated.mp3` | 脱落したとき | 残念系SE |
| `end.mp3` | 対局終了（全員終了） | 任意。締めの音 |
| `bgm.mp3` | 対局中ずっと（ループ） | BGM。下記参照。任意 |

## BGM（`bgm.mp3`）

- **ループ再生**されるので、ループしても不自然にならない素材を推奨。
- 既定は **OFF**。メニュー（左上 ☰）の「BGM（音楽）」で**端末ごとにON**にします
  （全端末で鳴らすと重なって濁るため、スピーカーのあるホストPCだけONにする想定）。
- 音量は効果音とは**別系統の専用スライダー**で調整します（既定は控えめ）。
- 未配置でもエラーにはならず、BGMだけ無音になります（後から置けば鳴ります）。

## 推奨仕様

- 形式: **mp3**（全ブラウザで再生可）
- 長さ: 効果音は **0.3〜2秒程度**と短めに（`applause`/`end` はやや長くてもOK）。`bgm` はループ前提で長さ自由
- 音量はファイル側で揃えておくと、端末側の音量スライダーで扱いやすいです
- すべての端末で同時に鳴るため、耳ざわりにならない自然な音量・音色を推奨

## 入手の目安

フリー素材（CC0 / 商用可）を利用する場合は、ライセンス表記を確認のうえ
