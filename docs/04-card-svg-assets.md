# 04. トランプSVG素材の生成

**フェーズ**: 1　**依存**: なし（01と並行可）　**対象**: `public/cards/`

## 概要
トランプカード52種＋裏面のSVG素材を自作する。ライセンス問題を回避し、シニア向けに数字・スートを大きく明瞭にデザインする。

## 関連要件
- `REQUIREMENTS.md` 4.3（カードデザイン）、6.5（グラフィック素材）、開発ロードマップ フェーズ1

## 仕様メモ
- カジュアルで親しみやすい雰囲気。**数字・スートを大きく**（シニアの視認性最優先）
- 赤（♥♦）・黒（♠♣）のコントラストをはっきり
- 命名規則：スペード `s1.svg`〜`s13.svg`（A〜K）/ ハート `h1`〜`h13` / ダイヤ `d1`〜`d13` / クラブ `c1`〜`c13` / 裏面 `back.svg`
- カードID（01で定義）と命名を一致させる
- 出せる札のハイライト／出せない札のグレーアウトは**UI側のスタイル**で表現する想定（SVG自体は素の状態でよい）

## 実装メモ
- 生成は `scripts/generate-cards.mjs`（依存追加なしのNode ESM）。再生成は `npm run cards:generate`。
- viewBox `0 0 250 350`。スート♠♥♦♣はSVG `<path>` で自作（フォント非依存）。
- デザイン: 四隅に大きなランク＋小スート（右下は180°回転）、中央に大スート1つ。絵札J/Q/Kは中央に大きな英字＋上に中スート。赤 `#d40000`／黒 `#1a1a1a` で高コントラスト。
- 命名は `src/lib/sevens/cards.ts` の `cardId`（例 `d7`）と一致。
