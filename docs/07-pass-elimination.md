# 07. パス回数・脱落管理

**フェーズ**: 2　**依存**: 03　**対象**: `src/lib/sevens/pass.ts`

## 概要
パス回数の上限管理と、上限超過時の脱落処理（手札を全て場に出す）を実装する。

## 関連要件
- `REQUIREMENTS.md` 3.2「パスのルール（重要）」、3.6

## 仕様メモ
- **パス可能回数はホストが部屋作成時に設定（1〜5回）**。ゲーム設定値として保持
- 残りパス回数は各プレイヤーごとに管理し、UIに常時表示（表示は 16/15 側）
- 設定回数を**超えてパス**した場合、そのプレイヤーは**手札を全て場に出して脱落**
  - 「手札を全て場に出す」処理の仕様を明確化（出せる札だけか、強制的に全部か）。標準解釈は**全部場に出して脱落＝以後の手番から除外**
- 脱落は順位（08）とは別枠で記録する
