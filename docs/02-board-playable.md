# 02. 場の管理・出せる札判定

**フェーズ**: 1　**依存**: 01　**対象**: `src/lib/sevens/board.ts`, `src/lib/sevens/playable.ts`

## 概要
場（4スートの並び）の状態管理と、ある手札から「今出せる札」を判定する純粋ロジックを実装する。

## 関連要件
- `REQUIREMENTS.md` 3.2（進行・スートの広がり）、7.1

## 仕様メモ
- 各スートは **7を起点**に、8→9→…→K の上方向と、6→5→…→A の下方向の2方向へ伸びる
- 場の状態はスートごとに「現在の上端ランク／下端ランク」（または並べた集合）で表現する
- 出せる札 = 各スートの上端+1 か 下端−1 に一致するカード。7はそのスートが未着手なら出せる
- 開始時：♦7（標準ルール）から、または各スートの7を場に置いて開始（`REQUIREMENTS.md` 3.2）。どちらの開始方式かは設定で切替できる設計が望ましい
