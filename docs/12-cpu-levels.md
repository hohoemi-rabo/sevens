# 12. CPU補完（弱・中・強）

**フェーズ**: 3　**依存**: 06　**対象**: `src/lib/sevens/cpu/{weak,medium,strong}.ts`

## 概要
CPUの思考を弱・中・強の3段階に作り分け、ホスト設定（11）で選択できるようにする。参加者が4人未満のとき自動でCPUを追加する。

## 関連要件
- `REQUIREMENTS.md` 3.4（CPUプレイヤー）

## 仕様メモ
- 06で定義した思考インターフェース `decide(state, playerId): Action` を3実装で差し替える
- **弱**：出せる札があればほぼ即出し。戦略性なし（生徒さんが勝ちやすい）
- **中**：自分の手札の連続性を見て、止めると有利な札は少し溜める
- **強**：相手の動きを読み、キーカード（端に近い札）を戦略的に止める
- 思考時間 1〜2秒の演出（待たせすぎない）
- 各CPUに名前・アバター付与（例「りつこ」「ハジメ」「ミミ」）。アバター素材の置き場所も決める
- CPUは人間と同じパス・脱落ルール（07）に従う
