# 01. カード定義・配札

**フェーズ**: 1　**依存**: なし　**対象**: `src/lib/sevens/cards.ts`, `src/lib/sevens/deal.ts`

## 概要
7並べの土台となるカードの型定義と、52枚を4人へ均等配分する配札ロジック（純粋TS）を実装する。通信・UIに依存しないこと。

## 関連要件
- `REQUIREMENTS.md` 3.1 / 3.2（基本ルール）、6.3、7.1（ゲームロジック層）

## 仕様メモ
- トランプ **52枚**（ジョーカーなし）。スート4種 × ランク A(1)〜K(13)
- 4人に均等配分（各13枚）
- スート識別子は SVG 命名と揃える: `s`(スペード) / `h`(ハート) / `d`(ダイヤ) / `c`(クラブ)
- カードは `{ suit, rank }` で表現し、`id`（例 `d7`）で一意化できると後段（場・手札・UI）で扱いやすい
