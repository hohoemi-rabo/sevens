# 08. 順位判定

**フェーズ**: 2　**依存**: 03, 07　**対象**: `src/lib/sevens/ranking.ts`

## 概要
手札を出し切った順に1〜4位を確定し、脱落者も含めた最終結果を算出する。

## 関連要件
- `REQUIREMENTS.md` 3.2「勝敗・順位（フル仕様）」、3.6

## 仕様メモ
- 手札を最も早く出し切った順に **1位・2位・3位・4位**
- **脱落者**（パス超過で手札放出）も明示的に表示。順位とは区別する
- 全員が上がるか脱落するまで継続
- 上がり/脱落の発生順を記録しておくと結果画面（17）で順位を組み立てやすい

## 実装メモ
- `src/lib/sevens/ranking.ts`（純粋・状態を変更しない派生ビュー）: `computeStandings(state): Standing[]` と `standingLabel()`。
- 並び順（ユーザー決定）: 上がりを `rank` 昇順 → 脱落者を `eliminatedOrder` 降順（後に脱落＝長く生存が上）→ 進行中は末尾。
- 脱落者は数字順位を付けず**「脱落」表示**（順位とは区別）。`Standing { player, outcome, rank? }`。
- `GameTable.tsx` の Results を `computeStandings` ベースに更新（脱落は「脱落」表示・赤系）。本格的な結果画面は17。
