# 16. PC横長レイアウト・シニアUI仕上げ

**フェーズ**: 4　**依存**: 05, 15　**対象**: `src/components/game/`, `src/components/ui/`, `src/app/globals.css`

## 概要
PC横長画面を最優先に対局画面を作り込み、シニア向けのUI配慮（大きな文字・ボタン、高コントラスト、誤操作リカバリ）を仕上げる。

## 関連要件
- `REQUIREMENTS.md` 4（画面レイアウト）、3.7（シニア向けUI配慮）、1.2

## 仕様メモ
- レイアウト（`REQUIREMENTS.md` 4.1）：上部に対戦相手3人（アバター・名前・残枚数・残パス回数）、中央に4スートの場（7を中央基準に左右へ）、下部に自分の手札、左下「パス（残りn回）」・右下「出す」・中央に自分のアバター、右上に音量・お助けトグル、左上メニュー
- 7を中央基準に A〜K が横一列に並ぶゆとりあるレイアウト
- マウス操作前提：クリックで選択、ホバーでハイライト
- シニア配慮（3.7）：
  - 大きなカード・ボタン（最小タップ領域 **60×60px以上**）
  - 高コントラスト（赤/黒スートをはっきり、数字大きく）
  - **「待って」ボタン**：自分のターンを一時停止
  - **出す前の確認**「このカードを出しますか？」（オンオフ可・デフォルトON）
  - 今は誰の番か・何をすべきかが一目で分かる明瞭なナビ
  - 読みやすい日本語・用語補足（例「パス（出せないとき）」）
- タブレットでも崩れないレスポンシブ対応
- Tailwind v3 を使用（`CLAUDE.md` 参照）
