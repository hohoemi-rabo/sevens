# 00. 開発チケット一覧（インデックス）

`REQUIREMENTS.md` と `CLAUDE.md` を元に、要件・機能ごとに分割した開発チケット集。各チケットは独立したファイルで、内部に Todo（`- [ ]`）を持つ。**完了したら `- [ ]` を `- [x]` に変更**して進捗を管理する。

## 進め方
- チケットは原則として番号順（＝依存関係順）に進める
- 各チケットの「依存」を確認し、前提が未完なら先にそちらを進める
- 1チケット完了ごとにコミットする（コンベンショナルコミット形式）
