# 15. お助けモード

**フェーズ**: 4　**依存**: 02, 05, 07　**対象**: `src/components/help/`

## 概要
初心者でも遊べるよう、出せる札のハイライトやパス警告などの補助を行う「お助けモード」を実装する。トグルで切替、**デフォルトはON**。

## 関連要件
- `REQUIREMENTS.md` 3.3（お助けモード）、1.2（初心者ファースト）

## 仕様メモ
- 画面上のトグルスイッチで切替（右上、`REQUIREMENTS.md` 4.1）
- **ON時**：
  - 出せる札をハイライト（02の `playableCards` を利用）
  - 出せる札があるのにパスしようとしたら確認ダイアログ「出せる札がありますが、パスしますか？」（07の判定を利用）
  - 残りパス回数の強調表示
  - 自分の番が来たことの明示（ターン通知）
- **OFF時**：自分の番の通知のみ（最低限）
- トグル状態は端末ごとに保持
